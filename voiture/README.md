# Voiture autonome

Arduino robot car with obstacle avoidance.

## Hardware / Materiel

* Robot Smart Car châssis Kits B073169GBF
* L298 N module contrôleur de Moteur double H-bridge B077LV4W5Z
* HC-SR04 B00PQB8GWM
* IR Infrared Obstacle Avoidance Sensor
* Arduino UNO

## PIN

* L298N IN1 <-> UNO D3
* L298N IN2 <-> UNO D4
* L298N IN3 <-> UNO D5
* L298N IN4 <-> UNO D6
* HC-SR04 TRIG <-> UNO D8
* HC-SR04 ECHO <-> UNO D9
* IR Avoidance <->  UNO D10
